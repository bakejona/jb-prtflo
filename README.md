# Jonathan Baker: UX/UI Designer & Developer Portfolio

## Features
Responsive Navigation: Includes a fixed navigation bar with smooth scrolling links to the main sections (Home/About, Projects, Contact).

About Section: Highlights skills including HTML, CSS, JavaScript, React.js, PHP, Figma, Adobe XD, and Adobe Illustrator.

Project Showcase: A dynamic grid displaying recent work with project descriptions, tech stack badges, and direct links to GitHub repositories and live application/demos.

Contact Form: A functional form for visitors to get in touch, utilizing the EmailJS service for secure communication.

Performance: Built with Next.js for optimization and includes Vercel Analytics for performance monitoring.

## Tech Stack
This project is built using the following core technologies and libraries:

### Frontend

Framework: Next.js

Library: React

Styling: CSS Modules, PostCSS, and Tailwind CSS

UI Icons: React Icons

### Form Submission

@emailjs/browser: For handling contact form submissions.

nodemailer: Listed as a dependency, likely used for server-side email functionality or reference.
